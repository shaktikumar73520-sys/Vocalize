Here is the complete list of all features (core + extra / advanced) from the Vocalize app specification. No code, just the features – exactly as requested.

---

✅ CORE FEATURES (Must-have for MVP)

Recording & Storage

1. Voice recording – tap to record, stop, preview, then save.
2. Internal storage – all .m4a audio files saved privately in app’s internal folder (no external permissions needed).
3. Auto‑naming – files named with timestamp + unique ID.
4. Metadata storage – title, duration, date created/modified, file path, etc., stored in Room database.
5. Edit title & text note – user can add/edit title and a plain text note for each memo.

Playback & Management

1. Playback controls – play, pause, seek, speed selection (0.5x – 2x).
2. Memo list – view all memos in a scrollable list (with pagination).
3. Recents tab – last 10 memos for quick access.
4. Memo card actions – ▶ Play, ⏰ Set reminder, ⋮ menu (edit title, delete, add to playlist).
5. Delete memo – removes audio file and metadata.
6. Share memo – export as audio file (.m4a) to other apps.

Reminders & Notifications

1. Set one‑time reminder – pick date & time, get notification to listen.
2. Repeat reminders – daily, weekly, or custom days (e.g., Mon/Wed/Fri).
3. Notification actions – Play (opens app & plays), Snooze (10 min), Dismiss.
4. Full‑screen reminder – (Android 10+) shows large player card when phone unlocked.
5. Calendar view – month grid with dots on days that have reminders. Tap a date to see memos scheduled.

Organization

1. Categories – assign a category (Work, Personal, Ideas, Shopping, Health) with colour & icon.
2. Playlists – create custom playlists, add memos, reorder, play entire playlist.
3. Play all – button on playlist card to play all memos sequentially.

Search & Filter

1. Search bar – search by memo title or text note.
2. Voice input for search – mic icon uses on‑device speech recognizer to fill search query.
3. Filters – by category, date range, has reminder (yes/no), playlist.

User Interface & Animations

1. Splash screen – logo + fade animation, preloads Vosk model in background.
2. FAB pulse animation – idle recording button slowly scales.
3. Waveform animation – real‑time microphone level during recording.
4. Card swipe to delete – swipe left/right on a memo card to delete.
5. Shared element transition – card expands smoothly into detail screen.
6. Micro‑interactions – play button rotates, seekbar spring effect, FAB morphs into stop button.
7. Dark / Light theme – follows system or manual override.
8. Accent colour picker – user can change app’s primary colour.

Data Management

1. Import audio – pick .m4a or .mp3 from device storage, copies into app.
2. Export memo – long‑press → share as audio or metadata text.
3. Backup to Google Drive – manual backup of database + all audio files (uses free AppDataFolder).
4. Restore from Google Drive – fetch latest backup, replace local data.
5. Storage info – show used space, “Clear cache” button.
6. Delete all data – option in settings.

Permissions & System Integration

1. Permission handling – request RECORD_AUDIO, POST_NOTIFICATIONS (Android 13+), SCHEDULE_EXACT_ALARM (Android 12+), USE_FULL_SCREEN_INTENT, FOREGROUND_SERVICE.
2. Background playback – uses foreground service with media controls.
